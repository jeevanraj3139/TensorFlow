This project uses an Artificial Neural Network (ANN) built with TensorFlow/Keras to predict the likelihood of a stroke based on patient health data.
Features :
Handles missing BMI values using mean imputation.
Encodes categorical features using Label Encoding.
Standardizes data using StandardScaler.
Splits data into training and testing sets.
Trains an ANN model for stroke prediction.
Visualizes training and validation accuracy/loss.
